CoupangOrder 주요 필드:
- id: String (UUID)
- companyId: String (UUID)
- shipmentBoxId: String (unique)
- orderId: String (VarChar30)
- totalPrice: Int
- status: String (ACCEPT/INSTRUCT/DEPARTURE/DELIVERING/FINAL_DELIVERY)
- orderedAt: DateTime
- orderItems: CoupangOrderItem[]

CoupangOrderItem 주요 필드:
- id: String (UUID)
- orderId: String (FK -> CoupangOrder.id)
- vendorItemId: String (VarChar30)
- vendorItemName: String
- sellerProductId: String? (VarChar30) -- 기존 productId(UUID)와 다름! 쿠팡 상품 ID.
- sellerProductName: String
- shippingCount: Int (기존 quantity 대신)
- salesPrice: Int
- orderPrice: Int
- instantCouponDiscount: Int

**핵심:** CoupangOrderItem에는 productId (UUID FK) 없음. sellerProductId (String, 쿠팡 ID)만 존재.
기존 Order.productId 기반 groupBy는 -> sellerProductId 기반 또는 orderCount=0 반환으로 처리.
