# 상품 파이프라인 시나리오

## 전제조건
- dev:all 실행 중
- 상품 데이터 존재

## 시나리오

### 1. 소싱 → 상품 등록
- `/sourcing` 접속 → 소싱 목록 로드 확인
- 상품 클릭 → 상세 페이지 이동
- 탭 전환 (기본정보, 이미지, 원본데이터)
- 에디터 페이지 접근 가능 확인

### 2. 상품 목록 CRUD
- `/products` 접속 → 페이지네이션 동작
- 필터 (등급, 상태, 검색) 동작 확인
- 상품 클릭 → 상세 페이지 이동
- 워크플로우 실행 버튼 표시

### 3. 상품 상세 페이지
- 기본 정보 카드 표시
- 탭 전환: 트렌드, 광고, 재고, 리뷰, 활동
- 워크플로우 메뉴 열기/닫기 동작

### 4. 썸네일 AI
- `/thumbnails` 접속
- 분류 완료/미분류 탭 전환
- KPI 카드 수치 표시
- 등급 도넛 차트 클릭 → 필터 연동

## 검증 기준
- [ ] 모든 페이지 정상 로드
- [ ] API 에러 없음
- [ ] 페이지 전환 시 로딩 스켈레톤 표시
- [ ] 데이터 없을 때 빈 상태 메시지 표시
