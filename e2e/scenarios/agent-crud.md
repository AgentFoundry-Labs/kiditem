# Agent OS CRUD 시나리오

## 전제조건
- dev:all 실행 중 (localhost:3000 + localhost:4000)
- marketplace 시드 데이터 존재 (`scripts/seed-marketplace.sql`)

## 시나리오

### 1. 마켓플레이스에서 에이전트 설치
- `/agents?tab=marketplace` 접속
- 에이전트 탭 필터 클릭 → 에이전트만 표시되는지 확인
- '매니저 에이전트' 카드 클릭 → 설치 모달 표시
- '고용하기' 버튼 클릭 → 성공 토스트 확인
- 마켓플레이스 카드에 '고용됨' 뱃지 표시 확인

### 2. 에이전트 목록 확인
- '에이전트 관리' 탭 클릭
- 설치한 에이전트가 목록에 표시되는지 확인
- 상태 표시 (idle/running/paused) 확인
- 필터 (전체/활성/일시정지/에러) 동작 확인

### 3. 에이전트 상세 페이지
- 에이전트 행 클릭 → `/agents/[id]` 이동
- 탭 전환: 대시보드, 인스트럭션, 스킬, 설정, 실행이력, 예산
- '뒤로' 버튼 → 목록으로 복귀

### 4. 에이전트 삭제
- 목록에서 삭제 아이콘 클릭
- confirm 다이얼로그 1회만 표시
- 삭제 후 목록에서 제거 확인
- 상세 페이지로 이동하지 않는지 확인 (과거 버그)
- 마켓플레이스에서 '고용됨' 뱃지 사라지는지 확인

### 5. 워크플로우 설치/해제
- 마켓플레이스 → 워크플로우 탭
- '주문 자동 처리' 설치
- 워크플로우 탭에서 표시 확인
- 마켓플레이스에서 해제 → '고용됨' 제거 확인

- [ ] 콘솔 에러 없음
- [ ] 설치/삭제 후 목록 즉시 갱신
- [ ] 마켓플레이스 installed 상태 동기화
