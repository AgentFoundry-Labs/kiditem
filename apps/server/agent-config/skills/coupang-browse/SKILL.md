# Coupang Browse Skill

## 주의사항

이 스킬은 Browse 도구가 허용된 에이전트만 사용 가능.
현재 대부분의 에이전트는 DRY-RUN 모드 (판단만, 실행 안 함).

## 쿠팡 Wing 대시보드

- URL: `https://wing.coupang.com`
- 로그인 필요 (셀러 계정)

### 광고 관리

```
wing.coupang.com → 광고 → 광고상품 관리
- 상품별 일일 예산 조정
- 키워드 입찰가 조정
- 광고 시작/중지
```

### 상품 관리

```
wing.coupang.com → 상품 → 상품 관리
- 가격 변경
- 재고 수량 변경
- 판매 시작/중지
```

### 주문 관리

```
wing.coupang.com → 주문 → 발주 확인
- 미출고 주문 확인
- 송장 입력
- 반품 승인/거부
```

## 안전장치

- 로그인 세션 만료 감지 → 즉시 중단, 결과 보고
- 3회 연속 실패 → 나머지 스킵
- 가격 변경 시 ±30% 이상 변동 금지
- 광고 예산 일일 상한 초과 금지
