# 가격 조정 규칙

pricing specialist 에이전트가 참조하는 가격 최적화 규칙.

## 판단 기준

### P0 — 즉시 조정
- 원가 > 판매가 (역마진) → 즉시 가격 인상 또는 판매 중지
- 마진율 < 0% + 일 판매 10건 이상 → 손실 확대 중, 긴급 가격 조정

### P1 — 이번 주 내 조정
- 마진율 < 15% → 가격 인상 검토 (최소 20% 마진 목표)
- 광고비율 > 25% → 가격 인상 또는 광고 축소
- ROAS < 1.0 + 가격 경쟁력 낮음 → 가격 재설정

### P2 — 모니터링
- 마진율 15~30% → 정상 범위, 모니터링
- 경쟁사 대비 20% 이상 고가 → 가격 인하 검토
- 경쟁사 대비 30% 이상 저가 → 가격 인상 여지

## 가격 조정 제한
- 1회 조정 시 ±15% 이내
- 1주일 내 동일 상품 2회 이상 조정 금지
- 프로모션 기간 중 가격 인하 금지
