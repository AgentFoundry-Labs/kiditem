# KIDITEM 광고 운영 원칙

이 문서는 광고 전략 에이전트의 판단 기준. 규칙 추가/수정 = 이 문서 수정.

## 일일 상한

- 총 광고비 상한: ₩500,000/일 (기본값, 셀러 설정 가능)
- 단일 상품 최대 예산 증가: 30%/일
- 상한 도달 시: 더 이상 예산 증가 금지, 감소만 가능

## 광고 중단 기준 (자동 실행)

| 조건 | 액션 | 우선순위 |
|------|------|----------|
| 재고 0 AND 광고 진행 중 | 즉시 중단 | P0 |
| 2주 이상 적자 + 광고비 집행 | 즉시 중단 | P0 |
| ROAS < 0.8 AND 7일 연속 | 광고 중단 | P1 |

- "7일 연속": 최근 7일 모두 ROAS < 0.8 (1일이라도 미달이면 리셋)
- "2주 이상 적자": 최근 14일 중 12일 이상 적자 (2일 허용)

## 광고 확대 기준 (상한 내)

| 조건 | 액션 |
|------|------|
| ROAS > 2.5 AND 7일 연속 | 예산 20% 증가 |
| A등급 + ROAS > 2.0 | 예산 30% 증가 |
| CTR > 5% AND 전환율 > 3% | 키워드 입찰 10% 인상 |

## 광고 축소 기준

| 조건 | 액션 |
|------|------|
| ROAS 1.0~1.5 AND 하락 추세 | 예산 20% 감소 |
| C등급 상품 | 예산 최소화 (₩1,000/일) |
| CTR < 0.5% | 해당 키워드 중단 |

## 등급 정의

- **A등급**: `abc_grade = 'A'` (매출 상위, 핵심 상품)
- **B등급**: `abc_grade = 'B'` (중간)
- **C등급**: `abc_grade = 'C'` (하위, 광고 최소화)

## 실행 후 기록

모든 조정은 이유와 함께 기록:
- 형식: `{상품명}의 {지표}가 {기준}에 해당 → {액션}`
- 예: "아기 젖병 세트의 ROAS 0.6이 7일 연속 0.8 미만 → 광고 중단"

## 안전장치

1. **일일 상한 체크**: 실행 전 오늘 총 집행액 조회 → 남은 예산 계산
2. **단건 상한 체크**: 단일 상품 예산 증가 30% 초과 → 스킵
3. **dry-run 모드**: 모든 액션을 JSON으로 출력만 (실행 안 함)
4. **롤백 기록**: 실행 전 상태 스냅샷 저장
