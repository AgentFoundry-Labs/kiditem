# 리뷰 모니터링 규칙

review_monitor specialist 에이전트가 참조하는 리뷰 분석 규칙.

## 판단 기준

### P0 — 긴급
- 최근 7일 1점 리뷰 3건 이상 → 품질 이슈 긴급 조사
- 평점 급락 (전월 대비 0.5 이상 하락) → 원인 분석 필요

### P1 — 개선 필요
- 평균 평점 < 3.5 → 리스팅/상품 개선 필요
- 리뷰 0건 + 판매 50건 이상 → 리뷰 유도 필요
- 부정 키워드 반복 (배송 지연, 파손, 다름) → 해당 이슈 대응

### P2 — 모니터링
- 평균 평점 3.5~4.0 → 개선 여지, 키워드 분석
- 리뷰 수 < 10건 + 판매 중 → 리뷰 축적 모니터링

### P3 — 양호
- 평균 평점 > 4.0 + 리뷰 수 > 20건 → 양호, 유지

## 키워드 분석

리뷰 본문에서 반복되는 키워드 추출:
- 긍정: 가성비, 만족, 빠른배송, 선물
- 부정: 파손, 다름, 늦음, 불량, 냄새

부정 키워드 3건 이상 반복 시 해당 카테고리로 액션 추천.

## 데이터 소스
- reviews: product_id, rating, content, created_at
- products: name, abc_grade, status
- orders: 판매 수량 (리뷰율 계산용)
